# Portfolio Website  

A personal portfolio website showcasing my profile, projects, and contact details. Built using **HTML** and **CSS** with a simple, responsive design.  

## 🌐 Features  
- **Home Page (`index.html`)** – Introduction and landing page.  
- **About Page (`about.html`)** – Details about me and my background.  
- **Projects Page (`projects.html`)** – A showcase of my work and projects.  
- **Contact Page (`contact.html`)** – A way for visitors to reach out.  
- **Custom Styling (`styles.css`)** – Unified theme and responsive design.  
- **Profile Image (`images/magi profile.png`)** – Personal branding.  

## 🚀 Getting Started  

### Prerequisites  
- A modern web browser (Chrome, Firefox, Edge, Safari).  
- (Optional) A local server if you want to extend with backend features later.  

### Running Locally  
1. Clone this repository:  
   ```bash
   git clone https://github.com/magibalan05/My_portfolio.git
   cd portfolio/exp_1
   ```
2. Open `index.html` in your browser.  

That’s it! 🎉
